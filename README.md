# 🕌 Smart Masjid Connect

A modern **Masjid Management System** built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
This project aims to digitalize mosque operations such as prayer time management, announcements, and community engagement.

---

## 🚀 Features

* 📅 Prayer Time Management
* 📢 Announcements System
* 📖 Daily Hadith Display
* 🛠️ Admin Panel for managing content
* 🎨 Responsive Frontend UI (React + Tailwind CSS)
* 🔗 RESTful API (Node.js + Express)
* 🗄️ MongoDB Database Integration

---

## ⚙️ Installation & Setup

### 1. Clone Repository

```
git clone https://github.com/UmairAhmad-dev/Smart-MasjidConnect.git
cd Smart-MasjidConnect
```

### 2. Install Dependencies

```
npm install
```

### 3. Setup Environment Variables

Create `.env` file:

```
MONGO_URI=your_mongodb_connection
PORT=5000
```

### 4. Run Backend

```
node server.js
```

### 5. Run Frontend

```
cd smart-masjid-connect
npm run dev
```
